# strapi-artboard

Strapi Artboard is a portfolio template for artists, I have created this project to better understand the Strapi CMS.


## Strapi Commands

  ### yarn develop
  Start Strapi in watch mode.

  ### yarn start
  Start Strapi without watch mode.

  ### yarn build
  Build Strapi admin panel.

  ### yarn strapi
  Display all available commands.

  ## React Commands

  ### npm start

  Start React server.
